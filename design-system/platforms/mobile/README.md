# Mobile

Componentes e padrões de layout para mobile, derivados do UI Kit
**iOS/iPadOS 27 (Community)**.

**Pendente:** o arquivo Figma de origem é um arquivo da Comunidade e a API
só expõe a capa. Duplique o arquivo para sua conta Figma
(https://www.figma.com/design/3vLCh8Rn06V4RyFyIXKyVS) e envie o novo link
para que os componentes reais (tab bar, nav bar, listas, controles) sejam
extraídos e documentados aqui.
